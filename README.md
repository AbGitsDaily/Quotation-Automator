# Quotation-Generator

Here I have created an app that automatically generates a quotation by just filling the details required to make a quotation.

You can enter 
1. Organisation name
2. Its GSTIN
3. Contact Number
4. Customer Name
5. Date of the Quotation
6. Add the Items
7. Add Advance Payment (if any)
8. Select Material Used
9. Enter about the material used
10. Select the Terms & Conditions
11. Preview the documnet
12. Download the document as a Word or PDF document.
