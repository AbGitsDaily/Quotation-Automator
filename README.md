# Quotation-Generator

Here I have created an app that automatically generates a quotation by just filling the details required to make a quotation.

You can enter 
1. Organisation name
2. Its GSTIN
3. Contact Number
4. Date of the Quotation
5. Add the Items
6. Add Advance Payment (if any)
7. Select Material Used
8. Enter about the material used
9. Select the Terms & Conditions
10. Preview the documnet
11. Download the document as a Word or PDF document.
