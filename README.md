# 📄 Quotation Automator

Generate professional business quotations in minutes with a simple web application built using Python and Streamlit.

This project was inspired by a real business workflow where creating quotations manually was repetitive, time-consuming, and prone to formatting inconsistencies. The application automates the entire process and produces clean, professional quotations that are ready to share with customers.

---

## 🌐 Try the application online

https://quotation-automator.streamlit.app/

---

## ✨ Features

- Generate professional quotations in seconds
- Browser-based application
- No installation required
- Customer and organisation details
- GSTIN support
- Material selection
- Itemized quotations
- Advance payment support
- Custom Terms & Conditions
- Live document preview
- Export as PDF
- Export as Microsoft Word

## 🚀 How It Works

1. Enter organisation details.
2. Enter customer information.
3. Add quotation items.
4. Select material used.
5. Configure payment details.
6. Choose Terms & Conditions.
7. Preview the quotation.
8. Download as Word or PDF.

## 💡 Why I Built This

Growing up around my family's business, I noticed that preparing quotations manually consumed unnecessary time.

Instead of repeatedly creating quotations from scratch, I wanted to build a simple application that automates the process while maintaining a clean and professional format.

This project combines automation with practical business needs.
